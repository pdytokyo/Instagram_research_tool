Instagram リサーチツール

リール動画を効率的に調査

Instagram リサーチツール - 使用ガイド

概要

Instagram リサーチツールは、Instagramリール動画を効率的に検索・分析するためのウェブアプリケーションです。ハッシュタグで検索し、いいね数、推定再生数、エンゲージメント率などの指標でリール動画を評価できます。

前提条件

このツールを使用するには以下が必要です：

  • Instagram Graph API のアクセストークン
  • Instagram ユーザーID
  • 結果保存用の Google スプレッドシート
  • Google Apps Script のデプロイURL

セットアップ手順

1. Instagram Graph API の設定
  1. Facebook Developer にアクセスし、アカウントを作成
  2. アプリを作成し、Instagram Basic Display を追加
  3. OAuth設定とアプリ検証を完了
  4. アクセストークンとユーザーIDを取得

詳細な手順については、Meta for Developers の公式ドキュメントを参照してください。

2. Google スプレッドシートの準備
  1. Google Drive で新しいスプレッドシートを作成
  2. スプレッドシートのURLからIDを取得
3. Google Apps Script の設定
  1. Google Apps Script にアクセス
  2. 新しいプロジェクトを作成
  3. デフォルトのコードを削除し、ツールの設定画面に表示されるコードを貼り付け
  4. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」を選択
  5. 必要な設定を行い、「デプロイ」ボタンをクリック
  6. デプロイされたウェブアプリのURLをコピー

ツールの使い方

初期設定
  1. 設定セクションに以下の情報を入力：
    • Instagram アクセストークン
    • Instagram ユーザーID
    • Google スプレッドシートID
    • Google Apps Script URL
  2. 「設定を保存」ボタンをクリック
リール動画を検索する
  1. 検索ボックスにハッシュタグを入力（「#」は不要）
  2. 必要に応じてフィルターを設定
  3. 「検索」ボタンをクリック
検索結果の見方

検索結果には以下の情報が表示されます：

  • サムネイル画像
  • 投稿日時
  • ユーザー名とフォロワー数
  • いいね数
  • 推定再生数（※実際の値ではなく推定値です）
  • いいね率と再生率
結果を保存する
  1. 検索結果ページの「スプレッドシートに保存」ボタンをクリック
  2. 指定した Google スプレッドシートに検索結果が保存されます

注意点

  • 再生数は推定値です：Instagram APIでは他アカウントの再生数を取得できないため、いいね数とフォロワー数を基に推定しています
  • API制限：Instagram APIには1時間あたりのリクエスト数に制限があります
  • ログイン済みユーザー向け：このツールはGraph APIアクセス権を持つユーザー向けです

トラブルシューティング

検索結果が表示されない

  • アクセストークンが有効か確認してください
  • ハッシュタグのスペルを確認してください
  • そのハッシュタグのリール動画が存在するか確認してください

スプレッドシートへのエクスポートが失敗する

  • スプレッドシートIDが正しいか確認してください
  • Google Apps ScriptのURLが正しいか確認してください
  • スプレッドシートへのアクセス権があるか確認してください

APIエラー

  • アクセストークンが有効期限切れでないか確認してください
  • API使用制限に達していないか確認してください
